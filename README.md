# React Weather App 🌤️

A modern and responsive weather application built with **React** and **Tailwind CSS**. Check real-time weather for any city in the world with a sleek and intuitive interface.  

---

## Features
- 🔍 Search weather by city name  
- 🌡️ View temperature, humidity, wind speed, and weather conditions  
- ☀️ Dynamic weather icons based on current conditions  
- 📱 Fully responsive design for desktop and mobile  
- ⚡ Fast and lightweight, styled with Tailwind CSS  

---

## Technologies Used
- **React** – Component-based front-end library  
- **Tailwind CSS** – Utility-first styling framework  
- **OpenWeatherMap API** – Fetches real-time weather data  
- **GitHub Pages** – For free hosting and deployment
